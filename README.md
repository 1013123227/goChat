goChat
======

This example application shows how to use WebSockets, the Go programming language and jQuery to create a simple web chat application. 

这个示例应用程序展示了如何使用 [WebSocket](http://tools.ietf.org/html/rfc6455), [Golang](http://golang.org/) 和  
[jQuery](http://jquery.com/) 创建一个简单的 web 聊天应用程序。这个示例的文章介绍见《[WebSocket 和 Golang 实现聊天功能](http://www.waylau.com/go-websocket-chat/)》 。
